\doxysubsection{Detailed Description}
Represents the \doxylink{class_bit_buddy}{Bit\+Buddy} pet in the game. 

\begin{DoxyAuthor}{Authors}
Ryan Hecht, Annabel Irani
\end{DoxyAuthor}
The \doxylink{class_bit_buddy}{Bit\+Buddy} has a set of attributes that can be updated by events. The \doxylink{class_bit_buddy}{Bit\+Buddy} can die if one of its attributes goes below zero. The \doxylink{class_bit_buddy}{Bit\+Buddy} can also purchase items and is born with a \doxylink{class_bit_buck_generator}{Bit\+Buck\+Generator} that generates Bit\+Bucks that are deposited into the user\textquotesingle{}s bank account.
