\doxysubsection{Detailed Description}
The \doxylink{class_light_button}{Light\+Button} class extends QPush\+Button to provide a light-\/themed toggle button. 

\begin{DoxyAuthor}{Author}
Shaylan Pratt
\end{DoxyAuthor}
This button is designed to toggle between two states, typically representing light and dark modes or turning a light-\/themed feature on and off. It emits signals upon changing themes or text modes, allowing for broader application-\/level changes in response to its state.
