\doxysubsection{Detailed Description}
The \doxylink{class_bit_buddy_status_widget}{Bit\+Buddy\+Status\+Widget} class is a widget that displays the status of the \doxylink{class_bit_buddy}{Bit\+Buddy}. 

\begin{DoxyAuthor}{Author}
Ryan Hecht
\end{DoxyAuthor}
The \doxylink{class_bit_buddy_status_widget}{Bit\+Buddy\+Status\+Widget} class is a widget that displays the status of the \doxylink{class_bit_buddy}{Bit\+Buddy}. Specifically, it displays the attributes of the \doxylink{class_bit_buddy}{Bit\+Buddy} along with their current values represented by how much each progress bar associated with the value is filled. For example, an attribute with a value of 2 out of 10 will have 20\% of the progress bar filled and an attribute that\textquotesingle{}s completely filled will have a comnpletely filled progress bar.
