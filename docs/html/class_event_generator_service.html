Detailed Description

The EventGeneratorService is responsible for generating events at random time intervals.

Author
Ryan Hecht

The EventGeneratorService is a singleton service that generates events at random time intervals and sends them to the EventDispatcherService to be dispatched to BitBuddy and throughout the entire application.
