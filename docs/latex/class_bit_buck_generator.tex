\doxysubsection{Detailed Description}
Generates Bit\+Bucks at a set interval and emits a signal when they are generated. 

\begin{DoxyAuthor}{Author}
Ryan Hecht
\end{DoxyAuthor}
A \doxylink{class_bit_buck_generator}{Bit\+Buck\+Generator} generates Bit\+Bucks at a set interval and emits a signal when they are generated. The number of Bit\+Bucks generated and the interval at which they are generated can be set (\textquotesingle{}upgrade\textquotesingle{}), but have default constants that they start at. The \doxylink{class_bit_buck_generator}{Bit\+Buck\+Generator} can be paused and started and is paused by default.
