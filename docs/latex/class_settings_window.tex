\doxysubsection{Detailed Description}
The \doxylink{class_settings_window}{Settings\+Window} class provides a dialog for users to adjust application settings. 

\begin{DoxyAuthor}{Author}
Shaylan Pratt
\end{DoxyAuthor}
This class offers a GUI for adjusting sound effects volume, music volume, and toggles to enable or disable sound effects and music. It interfaces with the application\textquotesingle{}s settings to persist these preferences.
