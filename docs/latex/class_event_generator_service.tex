\doxysubsection{Detailed Description}
The \doxylink{class_event_generator_service}{Event\+Generator\+Service} is responsible for generating events at random time intervals. 

\begin{DoxyAuthor}{Author}
Ryan Hecht
\end{DoxyAuthor}
The \doxylink{class_event_generator_service}{Event\+Generator\+Service} is a singleton service that generates events at random time intervals and sends them to the \doxylink{class_event_dispatcher_service}{Event\+Dispatcher\+Service} to be dispatched to \doxylink{class_bit_buddy}{Bit\+Buddy} and throughout the entire application.
