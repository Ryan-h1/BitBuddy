Detailed Description

Generates BitBucks at a set interval and emits a signal when they are generated.

Author
Ryan Hecht

A BitBuckGenerator generates BitBucks at a set interval and emits a signal when they are generated. The number of BitBucks generated and the interval at which they are generated can be set ('upgrade'), but have default constants that they start at. The BitBuckGenerator can be paused and started and is paused by default.
