Detailed Description

Container class that holds data surrounding the name of a BitBuddy attribute.

Authors
Ryan Hecht, Annabel Irani

Contains a UniqueName enum that represents the unique names of the BitBuddy attributes. Also features static methods to transform these names into string representations and to retrieve their corresponding images.
