Detailed Description

Defines the FileStorageService class, responsible for saving and loading BitBuddy objects.

Provides functionality to save the current state of a BitBuddy object to a JSON file and to load a BitBuddy object from a JSON file. It implements the Singleton design pattern to ensure only one instance of the service is created and used throughout the application.
