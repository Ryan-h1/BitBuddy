\doxysubsection{Detailed Description}
Service that handles all transactions in the game. 

\begin{DoxyAuthor}{Author}
Ryan Hecht
\end{DoxyAuthor}
This service is a singleton responsible for handling transactions between the user and the game. The approach with this service is questionable\+: events were originally supposed to be dispatched via the \doxylink{class_event_dispatcher_service}{Event\+Dispatcher\+Service} and nothing else, but this service causes some tighter coupling since transactions require confirmation by both parties to occur. If a refactor occurs, this service deserves to be scrutinized.
