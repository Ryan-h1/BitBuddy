\doxysubsection{Detailed Description}
The \doxylink{class_event}{Event} class represents an event that can be dispatched to listeners. 

\begin{DoxyAuthor}{Author}
Ryan Hecht
\end{DoxyAuthor}
The \doxylink{class_event}{Event} class represents an event that can be dispatched to listeners. Each event has a description and a creation time. Events are generally not meant to be constructed themselves as they do not provide much use. Instead, subclasses of \doxylink{class_event}{Event} are usually used.
