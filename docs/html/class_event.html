Detailed Description

The Event class represents an event that can be dispatched to listeners.

Author
Ryan Hecht

The Event class represents an event that can be dispatched to listeners. Each event has a description and a creation time. Events are generally not meant to be constructed themselves as they do not provide much use. Instead, subclasses of Event are usually used.
