Detailed Description

Represents the BitBuddy pet in the game.

Authors
Ryan Hecht, Annabel Irani

The BitBuddy has a set of attributes that can be updated by events. The BitBuddy can die if one of its attributes goes below zero. The BitBuddy can also purchase items and is born with a BitBuckGenerator that generates BitBucks that are deposited into the user's bank account.
