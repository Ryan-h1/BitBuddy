\doxysubsection{Detailed Description}
Defines the \doxylink{class_file_storage_service}{File\+Storage\+Service} class, responsible for saving and loading \doxylink{class_bit_buddy}{Bit\+Buddy} objects. 

Provides functionality to save the current state of a \doxylink{class_bit_buddy}{Bit\+Buddy} object to a JSON file and to load a \doxylink{class_bit_buddy}{Bit\+Buddy} object from a JSON file. It implements the Singleton design pattern to ensure only one instance of the service is created and used throughout the application.
