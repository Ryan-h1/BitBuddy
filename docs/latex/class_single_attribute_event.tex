\doxylink{class_single_attribute_event}{Single\+Attribute\+Event} is a subclass of \doxylink{class_event}{Event} that represents an event that modifies a single attribute of the \doxylink{class_bit_buddy}{Bit\+Buddy}. The event has an increment, an attribute, a probability, and a description.
