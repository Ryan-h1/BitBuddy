Detailed Description

The MusicService is responsible for playing and stopping the music in the game.

Author
Ryan Hecht

This service is a singleton that is responsible for playing and stopping the music in the game. It uses a QMediaPlayer in a loop to play music. It can only play one track in its current rendition.
