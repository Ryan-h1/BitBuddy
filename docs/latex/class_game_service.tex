\doxysubsection{Detailed Description}
The \doxylink{class_game_service}{Game\+Service} is responsible for starting and stopping the \doxylink{class_event_generator_service}{Event\+Generator\+Service} and \doxylink{class_music_service}{Music\+Service}. 

\begin{DoxyAuthor}{Author}
Ryan Hecht
\end{DoxyAuthor}
This service was originally meant to be a control paoint for every single service in the game. However, it now only controls the \doxylink{class_event_generator_service}{Event\+Generator\+Service} and \doxylink{class_music_service}{Music\+Service}. Refactoring is needed to make this class and its usage more sensible.
