Detailed Description

Provides functionalities for audio playback and control within the application.

The Audio namespace encapsulates methods for playing sound effects, initializing the audio system, muting/unmuting sound effects, and adjusting the overall volume. It's designed to offer a simple interface for managing sound effects within the application, leveraging Qt's multimedia capabilities.
